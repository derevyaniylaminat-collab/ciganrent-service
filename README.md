**Ciganrentauto** — це сучасна веб-платформа для швидкої та зручної оренди преміальних та повсякденних автомобілів. Проєкт реалізує повноцінний цикл оренди: від каталогу до панелі адміністратора.

## ✨ Основні можливості

- **Каталог автомобілів:** Зручний перегляд автопарку з фільтрацією за класом (Економ, Комфорт, Бізнес, Преміум), коробкою передач та статусом (вільне/зайняте).
- **Динамічний розрахунок вартості:** Калькуляція оренди "на льоту" з урахуванням каскадних знижок (5%, 10%, 15% за довгострокову оренду) та додаткових послуг (КАСКО, дитяче крісло, особистий водій тощо).
- **Особистий кабінет клієнта:** Можливість реєстрації, авторизації, перегляду активних оренд та історії замовлень.
- **Адміністративна панель:** Управління автопарком (можливість додавання та видалення машин), статистика доходів та зміна статусу замовлень.
- **Сучасний інтерфейс:** Адаптивний дизайн у стилі Glassmorphism з плавними мікроанімаціями.

## 🧰 Технологічний стек

- **Мова:** JavaScript (ES6+ Modules)
- **Збірка:** Vite
- **Стилі:** Vanilla CSS3
- **Зберігання даних:** LocalStorage API
- **Тестування:** Vitest

## 🚀 Запуск проєкту

Щоб розгорнути проєкт локально:

1. Встановіть залежності:
```bash
npm install
```
2. Запустіть локальний сервер:
```bash
npm run dev
```

Для збірки оптимізованої продакшн-версії:
```bash
npm run build
```

## 🧪 Автоматизовані тести

Проєкт покритий unit-тестами для перевірки логіки авторизації, обробки помилок та коректності фінансових розрахунків оренди.
Для запуску тестів виконайте команду:
```bash
npm run test
```
